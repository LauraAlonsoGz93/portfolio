Camito

It mimics like a real chameleon

This toy uses an Arduino motherboard and a RGB sensor fited on the bottom to catch the color of the surface where it is.

Interactive features

Camito includes some interactive features so the children can play and develop their psychomotor skills. These features are: color lights, vibration motor and sounds.

A friendly shape for the little children

Camito is a smiling baby chameleon. It has been designed with smooth surfaces, a big head and eyes, short legs and coiled tail. This details make children identify with the toy.
